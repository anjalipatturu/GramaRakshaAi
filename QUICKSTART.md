# GramaRaksha AI

Rural Healthcare Intelligence & Smart Triage System

## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run seed
npm run dev
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm start
```

Visit http://localhost:3000
